### 1.2 화면 목록

| # | 화면명 | 설명 | 우선순위 |
|---|--------|------|----------|
| 1 | 스플래시 | 앱 로딩 화면 | P1 |
| 2 | 온보딩 | 첫 사용자 안내 | P2 |
| 3 | 로그인 | 로그인/회원가입 | P1 |
| 4 | 홈 (대시보드) | 셀러 현황, 빠른 추천 | P1 |
| 5 | 스캔 | 카메라 스캔 화면 | P1 |
| 6 | 스캔 결과 | 인식 결과 확인/수정 | P1 |
| 7 | 내 셀러 | 와인 목록/검색/필터 | P1 |
| 8 | 와인 상세 | 개별 와인 정보 | P1 |
| 9 | 와인 등록/수정 | 와인 정보 입력 폼 | P1 |
| 10 | 추천 입력 | 음식/상황 입력 | P1 |
| 11 | 추천 결과 | AI 추천 와인 목록 | P1 |
| 12 | 설정 | 앱 설정 | P2 |
| 13 | 태그 관리 | 셀러/위치 태그 관리 | P1 |

| TasteProfile | 맛 프로필 표시 | 와인 상세 |
| Button | 버튼 (Primary/Secondary) | 전체 |
| Input | 입력 필드 | 로그인, 검색, 등록 |
| Modal | 모달/바텀시트 | 필터, 확인 |
| Toast | 토스트 알림 | 전체 |
